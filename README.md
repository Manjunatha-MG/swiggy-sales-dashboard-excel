# Swiggy Sales Dashboard (Excel Project)

## 📌 Overview
This project is an interactive **Swiggy Sales Dashboard** built using Microsoft Excel.

It provides insights into sales performance, customer ratings, and order trends.

---

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

---

## 🔑 Key Metrics
- Total Sales: $53.01M  
- Average Rating: 4.34  
- Avg Order Value: $268.51  
- Total Orders: 197K+  

---

## 📈 Features
- Monthly Sales Trend  
- Daily Sales Analysis  
- Weekly Sales Performance  
- Sales by Food Type (Veg vs Non-Veg)  
- State-wise Sales Analysis  
- Top 5 Cities by Sales  

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- KPI Cards
